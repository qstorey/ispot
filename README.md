# ispot

Convert iTunes playlists to Spotify playlists.

> WARNING: This project is a work in progress.

## Run

```bash
# Clone the repo
git clone https://github.com/qstorey/ispot

# Run
cd ispot
cargo run --
# or
cargo run -- --help
```

## Development

```bash
# Clone the repo
git clone https://github.com/qstorey/ispot

# Build a debug version
cd ispot
cargo build

# Run the unit tests
cargo test
```
